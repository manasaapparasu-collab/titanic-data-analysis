# Titanic Data Analysis 🚢

## 📌 Overview
This project analyzes the Titanic dataset to identify factors affecting passenger survival.

## 🧹 Data Cleaning
- Removed Cabin column (many missing values)
- Filled missing Age with mean
- Filled missing Embarked with mode
- Dropped unnecessary columns

## 📊 Analysis
- Survival rate by Gender
- Survival rate by Passenger Class
- Survival rate by Embarkation port
- Age distribution

## 🔍 Key Insights
- Females had higher survival rate (~74%)
- First class passengers survived more
- Higher fare increased survival chances

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
